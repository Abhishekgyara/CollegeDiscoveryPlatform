# 🎓 College Discovery Platform

A modern frontend web application that helps students discover, compare, and save colleges based on key factors like fees, ratings, placements, and location.

Built with a focus on clean UI/UX, responsive design, and real-world user experience for students making informed college decisions.

---

## 🚀 Features

### 📚 College Listing + Search
- Search and browse colleges easily
- Filter colleges by key parameters
- Pagination / Infinite scroll support for better UX

Each college card includes:
- College name
- Location
- Fees structure
- Rating

---

### 🏫 College Detail Page
A detailed view of each college with structured information:

- Overview of the college
- Available courses
- Placement statistics
- Student reviews (UI-ready structure)

---

### ⚖️ Compare Colleges
Side-by-side comparison feature for 2–3 colleges to simplify decision-making.

Comparison includes:
- Fees comparison
- Placement statistics
- Ratings
- Location

---

### 🔐 Authentication + Saved System (UI/Frontend Ready)
User-focused features:

- Login / Signup UI
- Save favorite colleges
- Save comparisons
- Personalized dashboard layout

---

## 🎯 Project Objective
This project was built to simulate a real-world SaaS product where students can:
- Discover colleges easily
- Compare options visually
- Save preferences for later review

---

## 🛠️ Tech Stack
- Next.js (React Framework)
- Tailwind CSS
- Lucide Icons
- Framer Motion (for animations)
- Context API (Auth state management)
